What can you do with these programs?

As an example of what it is all about, I want to take a closer look at the program 'Encyclopedia of Middlegame'. You can see a screenshot of this program beneath.

There is several things you can do with this program, which main purpose is to train the student in handling 'typical pawn structures' and 'openings and middlegame'. First of all you start to create yourself as a new user, and after this the program remembers which exercises etc. you went through so far. This is a good feature both for the student and the coach who can watch how his students are doing in the different openings. After creating a new user you start studying an opening as you can see from the screenshot above.
